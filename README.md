# Miller Wedding Website — v0.2

## Preview locally
Open `index.html` in a browser.

## Upload to GitHub
Upload the CONTENTS of this folder to the root of your GitHub repository:
- index.html
- styles.css
- script.js
- assets/

Then enable GitHub Pages:
Settings → Pages → Deploy from branch → main → /(root)

## Current scope
This version focuses on:
- a tactile, luxury envelope scene
- forest-green N&Z wax seal
- opening animation
- invitation reveal
- first transition into the wedding weekend site

## Notes
Google Fonts are loaded from the web. The site itself otherwise uses local assets.
